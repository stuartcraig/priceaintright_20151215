This directory contains the replication files for "The Price Ain't Right?
Hospital Prices and Health Spending on the Privately Insured," by Zack Cooper,
Stuart Craig, Martin Gaynor, and John Van Reenen. These files begin with the
existence of procedure level extracts. For information about obtaining these
files for replication, contact the Health Care Cost Institute (HCCI).

HC_var.do is the main control file.
